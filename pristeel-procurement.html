API Configuration
Platforma përdor Groq API (falas, pa kartë) për të analizuar dokumentet. Regjistrohu te console.groq.com → API Keys → krijo çelës. Çelësi ruhet vetëm në browser-in tënd.
GOOGLE DRIVE (opsionale)
Për të marrë dokumente direkt nga Drive. Krijo falas te console.cloud.google.com: aktivizo Google Drive API + Google Picker API → Credentials → krijo API Key dhe OAuth Client ID (Web). Te "Authorized JavaScript origins" shto: https://lemonstudiosks-collab.github.io
